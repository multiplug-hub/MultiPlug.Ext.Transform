# MultiPlug.Ext.Transform

This MultiPlug Extension allows values within MultiPlug Events to be transformed and published into new Events.

Supported Transformation Technologies:

1. Strings
	- To Upper Case
	- To Lower Case
	- Insert
	- Trim
	- Trim End
	- Trim Start
	- Pad Left
	- Pad Right
	- Remove
	- Replace
	- Substring

2. Regular Expressions

3. XSLTs
